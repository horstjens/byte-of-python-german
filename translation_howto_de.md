# Übersetzungsanleitung 

1. Der vollständige Quellcode des Original-Buches ist unter <https://github.com/swaroopch/byte-of-python> verfügbar.

2. Bitte [forken Sie das Repository](https://help.github.com/articles/fork-a-repo). (Legen Sie sich bei Github einen Account an und klicken Sie auf den "Fork" Button open rechts)

3. Laden Sie das Repository anschließend auf Ihren Computer herunter. Sie benötigen dafür Kenntnisse in der Verwendung von [Git](http://www.git-scm.com) (bzw. Sie klicken auf den grünen "Code" sodaß er aufklappt und dann auf "Download as zip")

4. Bearbeiten Sie die `.md`-Dateien, um sie in Ihre Sprache zu übersetzen. Idealerweise übersetzten Sie auch die `.py` und die `.txt`-Dateien im Unterordner `programs`

5. Aus den übersetzten Dateien bilden Sie eine vollständige Website, ein pdf oder ein Ebook im Epub Format. Dies ist auf mehrere Arten möglich, z.B. mittels [Honkit](https://honkit.netlify.app) oder [Sphinx](https://www.sphinx-doc.org)

6. Informationen zum Generieren der Website, des PDFs und des EPUBs finden Sie in der Datei [INSTALL.md](https://github.com/swaroopch/byte-of-python/blob/master/INSTALL.md).

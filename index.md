
# Byte of Python deutsch

"A Byte of Python deutsch" ist eine deutsche Übersetzung des englischsprachigen Buchs "[A byte of Python](https://python.swaroopch.com/)" von [Swaroop Chitlur](https://www.swaroopch.com/). "A byte of Python" ist eine Einführung in die Programmiersprache [Python](https://python.org).

:::{admonition} Über diese Übersetzung
:class: info
<details><summary>hier klicken</summary>

Diese Übersetzung beruht auf der englischen Originalausgabe von [A byte of Python](https://python.swaroopch.com/) (Stand Mai 2026), welche sich auf Python Version 3 bezieht. 

Ich werde versuchen die Übersetzung laufend aktuell zu halten, da sich auch das Originalprojekt (siehe Github Link) laufend ändert. 

Ich habe mir bei der Übersetzung kleinere Freiheiten erlaubt und nicht immer wortwörtlich übersetzt, z.B. wurden Textstellen die sich nur auf die veraltete Version 2 von Python beziehen gekürzt etc. 

Das Kapitel [Quiz](./quit_de.md) stammt rein vom Übersetzter. 

</details>

:::
